# Go で Momento Cache を始める

GoとMomento Cacheをすぐに使い始める必要がある場合、このページには必要な基本的なAPIコールが含まれています。[Go SDK のサンプル](https://github.com/momentohq/client-sdk-go/tree/main/examples) を参照してください。

## Momento SDKのインストール

[Go プロジェクトを作成](https://go.dev/doc/code#Command)したら、Momento Go SDKをインストールします。

```cli
go get github.com/momentohq/client-sdk-go
```

## API キーの設定

Momentoで認証するには、[Momento APIキー](/cache/develop/authentication/api-keys.md)が必要です。 [Momento Web Console](https://console.gomomento.com/caches)から取得できます。
トークンを取得したら、Momento クライアントが利用できるように環境変数に保存します：

```
export MOMENTO_API_KEY=<your Momento token here>
```

**Note**: セキュリティ強化のためには、トークンを環境変数ではなく、AWS Secret ManagerやGCP Secret Managerのようなものに格納するのがベストプラクティスだが、ここではデモのために使用しています。

## ライブラリのインポートと CacheClient オブジェクトの作成
最初に、必要なインポートを取り込み、メイン関数を設定します。


```go
package main

import (
	"context"
	"errors"
	"log"
	"time"

	"github.com/google/uuid"
	"github.com/momentohq/client-sdk-go/auth"
	"github.com/momentohq/client-sdk-go/config"
	"github.com/momentohq/client-sdk-go/config/logger"
	"github.com/momentohq/client-sdk-go/momento"
	"github.com/momentohq/client-sdk-go/responses"
)

func main() {
    fmt.Println("Hello, world.")
}
```
