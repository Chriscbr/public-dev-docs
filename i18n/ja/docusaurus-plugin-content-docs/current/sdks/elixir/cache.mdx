# Elixir で Momento Cache を始める

ElixirとMomento Cacheをすぐに使い始める必要がある場合、このページには必要な基本的なAPIコールが含まれています。[Elixir SDK のサンプル](https://github.com/momentohq/client-sdk-elixir/tree/main/examples) を参照してください。

## Momento SDKのインストール

最新版のMomento Elixir SDKは[Hex](https://hex.pm/packages/gomomento)にあります。

## APIキーの設定

Momentoで認証するには、Momento認証トークンが必要です。 [Momento Web Console](https://console.gomomento.com/caches)から取得できます。
トークンを取得したら、Momento クライアントが利用できるように環境変数に保存します：

```
export MOMENTO_API_KEY=<your Momento token here>
```
