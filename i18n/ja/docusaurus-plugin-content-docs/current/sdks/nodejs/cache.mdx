# JavaScript で Momento キャッシュを始める

Momento は 2 つの JavaScript SDK を提供します。 [Node.js 用に 1 つ](/sdks/nodejs) と [ブラウザーその他の Web アプリケーション用に 1 つ](/sdks/web)。 2 つの SDK は同一の API を備えているため、`import`ステートメントを除いてコードは同じに見えますが、内部では、異なる JavaScript ランタイム環境で最適なパフォーマンスと互換性を実現するように構築されています。

このページには、Momento Cache をすぐに使い始めるために必要な基本が含まれています。さらに詳しい情報と例については、上にリンクされている SDK ページを参照してください。

## Momento SDKをインストールする

既存の Node.js プロジェクトに Momento Node.js SDK をインストールするには:

```cli
npm install @gomomento/sdk
```

または、既存のブラウザ アプリケーション プロジェクトに Momento Web SDK をインストールするには:

```cli
npm install @gomomento/sdk-web
```

:::tip
2 つのライブラリのうち 1 つだけが必要です。ターゲット プラットフォームに応じて、`@gomomento/sdk` または `@gomomento/sdk-web` のいずれかです。両方は必要ありません。
:::

## API キーを設定する

Momento で認証するには、Momento API キーが必要です。 [Momento Web コンソール](https://console.gomomento.com/caches) から入手できます。
API キーを取得したら、Momento クライアントがそれを使用できるように環境変数に保存します。

```
export MOMENTO_API_KEY=<your Momento API key here>
```

**Note**: セキュリティを強化するには、API キーを環境変数ではなく AWS Secret Manager や GCP Secret Manager などに置くのがベスト プラクティスですが、ここではデモ目的で使用しています。
